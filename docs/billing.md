# 결제 및 구독 정책

쏠쏠 플랫폼의 구독 플랜, 결제, 청구, 환불에 관한 정책 문서.
결제/구독/환불 관련 페이지 구현 시 이 문서를 참조.

---

## 1. 플랜 구조

### 1.1 기본 원칙

- 모든 구독 플랜은 **정액제**
- 플랜별 기본 사용량 제공
- 초과 사용량 발생 전 플랜 변경 유도

### 1.2 무료 체험

- 플랜 사용 전 **14일 무료 체험** 제공
- 무료 체험 신청에도 **카드 등록 필수**
- 무료 체험 기간 완료 후 구독 취소를 누르지 않으면 해당 플랜 자동 결제

### 1.3 사용량 체크 및 제한

- 기본 사용량 초과 예정 시 이메일 알림 및 Customer Admin 로그인 시 팝업 Alert
  - **80% 초과 시**, **90% 초과 시** 플랜 Upsell 유도
- Customer Admin 상품/콘텐츠 제한에 따른 Validation check

**Alert 예시:**

| 상황 | 메시지 |
|------|--------|
| 동영상 저장 용량 초과 | "제공된 용량이 초과하여 업로드할 수 없어요 ㅠㅠ" |
| 학생/회원 수 제한 초과 | "회원 수가 초과하여 회원관리가 어렵네요. 플랜을 업그레이드 하세요. 플랜 변경 바로가기" |

## 2. 결제 상태

| 상태 | 설명 | 사용자 노출 |
|------|------|-------------|
| 결제 대기 | 청구서 생성, 결제일 미도래 | O |
| 결제 중 | PG로부터 응답을 받지 못한 상태 | X |
| 결제 완료 | 청구서 금액이 정상 결제된 상태 | O |
| 결제 실패 | 카드 만료, 잔고 부족 등으로 즉시결제 또는 자동결제 실패 | O |
| 결제 유예 | 빌링정보로 결제 요청했으나 자동결제 실패 상태 | O |
| 미결제 | 결제 유예에서 결제되지 않아 상품 해지(만료) 후 정상 결제 불가 | O |

---

## 3. 기간 정의

### 3.1 구독기간

- **정의**: 결제일 ~ 다음 결제일 전일
- 서비스 이용이 가능한 기간
- 종료일 자정(0시 10분) 기준으로 사용량 체크
- 구독 취소 예정이 없는 한 정기결제일(결제시점)까지 사용 가능

**예시:** 2025.03.10 첫 결제 → 구독기간: 2025.03.10 ~ 2025.04.09

### 3.2 청구기간

- **정의**: 결제일 ~ 다음 결제일 전일 (월/연 단위)

### 3.3 정기결제

- 정해진 시간(예: 결제일 오전 7시)에 자동 결제 처리
- 결제 직전에 청구서 생성

---

## 4. 청구서

### 4.1 생성

- **시점**: 정기결제일 (예: 결제일 전일 9시에 결제대기 상태 생성)
- 청구서 생성 시점에는 메일링 없음
- **구성**: 당월 구독플랜 요금

### 4.2 미납금액 처리

- 청구서 생성 시점에 계정의 미납금액이 존재하면 청구서 내역에 포함하여 **합산 결제**

**예시 (정상):**

```
플랜 요금 (Starter): 50,000원
구독기간: 2025.04.10 ~ 2025.05.09
총결제금액: 55,000원 (부가세 포함)
```

**예시 (미납 포함):**

```
플랜요금: 300,000원
(미결제) 초과요금: 74,000원
총결제금액: 411,400원 (부가세 포함)
```

## 5. 결제

### 5.1 카드 등록

- 카드 등록 단계부터 **PG(토스페이먼츠)** 결제창에서 진행
- 참고: [토스페이먼츠 빌링 결제창 연동](https://docs.tosspayments.com/guides/v2/billing/integration)

### 5.2 카드 결제

- **지원**: 신용카드, 체크카드
- 구독플랜의 최초 결제 및 정기 결제 요청 수단
- 정기결제일에 해당하는 날짜가 존재하지 않으면 **해당 월 말일에 결제** (예: 1/31 → 2/28)
- 이전 사용 플랜의 미납금액이 존재하면 결제 시 함께 결제

### 5.3 결제수단 변경

- 등록된 신용카드 또는 체크카드 변경 가능
- **결제일 당일에는 변경 불가**

### 5.4 결제 유예 기간 내 즉시결제

- 청구서(결제 실패 상태)에서 즉시결제를 선택하여 미결제 금액 결제(자동결제) 가능

### 5.5 계좌이체

- **미지원** (토스페이먼츠 계좌 기반 정기결제 미지원)

---

## 6. 영수증

- PG의 **신용카드 매출전표** 출력 연동
- 신용카드 매출전표가 세금계산서를 대체
- 확인 경로: `계정관리 > 결제내역 > 청구서`
- 참고: [토스페이먼츠 매출전표 가이드](https://docs.tosspayments.com/resources/glossary/sales-statement)

---

## 7. 메일링

### 7.1 결제 완료 안내

- 결제일에 청구서 금액 결제 처리 후 발송
- 내용: 결제된 내역, 해지 방법 안내 포함

### 7.2 결제 실패 안내

- 자동결제 실패 사실, 재결제 진행 방법, 결제 유예 정책 등 안내
- 기본 사용량 초과 사실 및 추가 요금 청구 예정 안내

---

## 8. 이용 정책

### 8.1 월간 구독

- 매월 플랜 요금 결제
- 다음 달부터 최초 결제일과 동일한 날짜에 자동 결제

### 8.2 연간 구독

- 12개월 치 요금 일괄 결제
- 다음 해부터 최초 결제일과 동일한 날짜에 매년 자동 결제

### 8.3 플랜 변경

- 서비스 중단 없이 플랜 갱신
- **조건**: 다른 구독 플랜 또는 결제 기간으로 변경 (예: Starter → Standard)
- **적용**: 즉시 적용. 결제 즉시 새로운 구독기간 시작, 청구 주기 변경
- 이전 구독플랜의 사용금액이 남아 있으면 **일할 계산하여 차액 제외** 후 결제금액 산정

### 8.4 구독 주기(결제 기간) 변경

| 변경 | 처리 방식 |
|------|-----------|
| 월간 → 연간 | 결제 즉시 새로운 구독기간 시작. 월간 미사용 기간 일할 계산하여 차액 제외 |
| 연간 → 월간 | 즉시 변경 불가. 연간 구독기간 종료 후 월간 구독 시작. 고객 컴플레인 시 수동 환불 처리 |

### 8.5 결제일 변경

- 별도 변경 수단 미제공
- 변경하려면 **플랜 해지 후 원하는 날짜에 재가입** 필요

## 9. 결제 실패 처리

### 9.1 최초 결제 실패 (3일간 재시도)

- 정기결제일 포함 **3일 동안 하루 2회씩 총 6회** 재결제 시도
- 재결제 시도 기간 동안 **정상적인 서비스 이용 가능**
- 상태: `사용중` (청구서: `결제실패`)

### 9.2 3일간 재결제 실패 (결제유예)

- **결제유예** 상태로 전환
- Customer Admin 서비스 이용 중지, **Front는 이용 가능**
- Customer Admin창에 콘텐츠 영역 닫힘 + Brand 사이트에서 추가결제 진행 유도 메시지 노출
- 언제든지 **즉시결제 가능**, 결제 시 해당 일 기준으로 구독기간 재시작
- 상태: `중지` (청구서: `결제유예`)

### 9.3 결제유예 상태 미결제 (상품 해지)

- 결제유예 기간 내 미결제 시 다음 정기결제일(월 기준)에 구독플랜 해지
- **결제 유예 기간: 27일**
- 일정 기간 경과 후 데이터 삭제 시 복구 불가
- 상태: `만료` (청구서: `미결제`)
- 미납금액은 별도 저장, **5년 경과 시 미청구금으로 변경**

## 10. 구독 해지

### 10.1 남은 구독기간 종료 후 해지

- **경로**: `홈페이지 > 이용상품정보 > 구독 취소 > 남은 구독기간 종료 후`
- 사용 중인 구독기간 동안은 계속 플랜 이용 가능 (기간 만료 후 해지)
- 구독 취소 후 마지막 청구기간 내 모든 요금 청구 후 등록된 카드 정보 자동 삭제

### 10.2 즉시 취소 및 환불

- **경로**: `홈페이지 > 이용상품정보 > 1:1 문의 > 구독 취소 > 즉시 취소 및 환불`
- 서비스 즉시 중지 및 환불 요청 **수동 접수** (환불계좌번호 요청)
- 관리자 확인 후 전액/부분 환불 처리
- 플랜 만료 후 새로운 구독플랜 구매 가능

---

## 11. 해지/환불 정책

### 11.1 무료 플랜

- 정기 구독 시작 전 취소 시 즉시 삭제 (결제 내역 없음)
  - 무료 수수료 정책이 있을 시 1개월 후 정산 후 즉시 삭제
- 무료 체험 후 자동 결제 시 원칙적으로 전액 환불 없음
  - 구독 취소 시 즉시 취소 및 환불 프로세스 진행

### 11.2 사용 중인 플랜

- **자동 환불 불가**, 관리자 확인 후 환불 진행

### 11.3 월간 구독 환불

| 조건 | 환불 |
|------|------|
| 사용 이력 없고 결제 후 7일 이내 | **전액 환불** |
| 그 외 | **부분 환불**: 잔여기간 일할 계산 후 위약금 30% 제외 |

**월간 환불 공식:**

```
환불금액 = [결제금액 – (월간 제공금액 ÷ 30일 × 사용일수)] × 0.7
```

### 11.4 연간 구독 환불

| 조건 | 환불 |
|------|------|
| 사용 이력 없고 결제 후 7일 이내 | **전액 환불** |
| 그 외 | **부분 환불**: 할인 전 금액 기준 일할 계산 후 위약금 30% 제외 |

- 해지 신청일이 포함된 월의 구독 기간 종료일까지 서비스 이용 가능
- 할인 적용 시 **정상금액 기준**으로 사용요금 제함

### 11.5 주의사항

- 미결제 상품 만료 후 재구매 시, 이전 미납금액이 존재하면 **함께 청구/결제**

## 12. 브랜드 결제페이지 안내문구

### 구독서비스 결제 안내

- 월간 구독: 매월 처음 결제한 날짜와 같은 날 자동 결제
- 연간 구독: 매년 처음 결제한 날짜와 같은 날 자동 결제
- 특정 월에 해당 날짜가 없으면 그 달의 마지막 날에 결제 (예: 3/31 → 4/30)

### 취소/환불 안내

- 언제든 취소 가능, 다음 결제일 이전 취소 시 익월부터 미결제
- 전액환불: 7일 이내 + 미사용 시 가능
- 남은 기간 종료 후 해지 시: 서비스 계속 이용 가능
- 즉시 취소 시: 서비스 즉시 중단

### 결제 증빙

- 신용카드 매출전표가 세금계산서를 대체
- 확인 경로: `계정관리 > 결제내역 > 청구서`
